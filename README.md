# Voluntário - Feira de Profissões do Cursinho Ideal - Unesp

Palestra realizada na Feira de Profissões do Cursinho Ideal apresentando o curso de Ciência da Computação.
O Cursinho Ideal é um projeto de extensão universitária destinado a alunos carentes de Presidente Prudente e região com o objetivo de fornecer preparação para o vestibular.
